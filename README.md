Personal portfolio site — plain HTML/CSS/JS, hosted on GitHub Pages.

**Live:** [markchauhan.github.io](https://markchauhan.github.io)

## Structure

```
├── index.html
├── css/
│   └── style.css
├── js/
│   └── main.js
└── .github/
    └── workflows/
        └── deploy.yml
```

## Deploy

Pushes to `main` auto-deploy via GitHub Actions. No build step needed.

To set up:
1. Push this repo to `github.com/markchauhan/markchauhan.github.io`
2. Go to **Settings → Pages**
3. Set source to **GitHub Actions**
4. Done — it'll be live at `markchauhan.github.io`
